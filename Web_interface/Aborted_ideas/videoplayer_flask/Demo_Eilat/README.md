# Indications

### To run the demo

1. Navigate to the directory:

```bash
cd Video_against_spectro/Demo_Eilat
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

or 

```bash
pip3 install -r requirements.txt
```

3. Run the app locally:

```bash

python app_demo_eilat.py
```
